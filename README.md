Leaf
=======

Leaf is a simple tiled Window Manager using XCB

## Requirements

* libxcb-devel >= 1.9-3

## How to

### Debug mode

<pre>
make
</pre>

### Release mode

<pre>
make DEBUG=0
</pre>
